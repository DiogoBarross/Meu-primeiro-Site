Tags Essenciais

Quais são elas?

  • Tag < html >
  • Tag < head >
  • Tag < title >
  • Tag < body >
  • Tag < a >
  • Tag < img >

Tag < html >

(Voltar)

Esta é a tag de abertura que define o início de um documento HTML. Todo o conteúdo do documento HTML é colocado entre as tags < html> e < /html>. É a raiz do documento HTML.

Tag< head>

(Voltar)

Esta tag contém informações sobre o documento HTML que não são exibidas diretamente no navegador, como metadados, links para estilos e scripts, definições de caracteres e outros elementos importantes para a formatação e comportamento do documento.

Tag< title>

(Voltar)

Esta tag é usada para definir o título do documento HTML. O texto especificado dentro da tag < title > é exibido na barra de título do navegador ou na guia. É uma parte crucial da otimização para mecanismos de busca (SEO) e ajuda os usuários a identificar o conteúdo da página.

Tag< body>

(Voltar)

Esta tag contém todo o conteúdo principal do documento HTML, incluindo texto, imagens, vídeos, links e outros elementos que são visíveis para o usuário no navegador. É dentro da tag < body> que a maior parte do conteúdo do site é inserida.

Tag< a>

(Voltar)

A tag é usada para criar links entre diferentes páginas da web, arquivos, seções da mesma página, endereços de email e outros recursos na internet. É uma das tags mais fundamentais para a construção de hipertexto na web.

Tag< img>

(Voltar)

Esta tag é usada para inserir imagens no documento HTML. O atributo src é usado para especificar o caminho para a imagem, enquanto outros atributos como alt são usados para fornecer texto alternativo para acessibilidade e SEO.
